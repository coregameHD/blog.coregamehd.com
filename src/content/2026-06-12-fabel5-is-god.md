ช่วงนี้ Fabel 5 เปิดให้ใช้ฟรีจนถึงวันที่ 22 มิถุนายน ผมเลยทดลองใช้สักหน่อย ผลปรากฏว่าเทพสมคำร่ำลือ ทำงานได้ดีจนน่าทึ่ง ยอมรับเลย

————————————
โจทย์: ผมต้องการระบบอัตโนมัติที่ช่วยดึงโพสต์จากเพจ Kagami Nihongo มาเก็บไว้ที่ฐานข้อมูล จากนั้นแปลงข้อมูลดิบให้อยู่ใน Schema ตามที่กำหนดไว้ เพื่อนำไปแสดงข้อมูลบนเว็บไซต์ ซึ่งตอนนี้ผมเขียนสคริปเชื่อม API จาก Facebook Page สำเร็จแล้ว

ปัญหา: การแปลงข้อมูลดิบ 3,000 กว่าโพสต์ที่ไม่มีรูปแบบตายตัวแถมยังมีทั้งภาษาไทย+อังกฤษ+ญี่ปุ่นปนกันไม่ใช่เรื่องง่าย ต้องเขียน condition ดักข้อมูลซ้อนกันหลายชั้น นอกจากนี้ยังมีรายละเอียดปลีกย่อยอย่างเช่นพิมพ์ผิด มีช่องว่างเกินมา รวมถึงปัญหาจุกจิกอีกเพียบ

————————————
ผมเลยโยนโค้ดทั้งหมดให้ Fabel ไปทำต่อ ผลปรากฏว่ามันทำให้ผมเสร็จหมดเลยภายในเวลาไม่กี่นาที วิเคราะห์และจัดกลุ่มให้เอง แยกแยะบริบทเป็นด้วย ลดงานไปได้เยอะมาก

เรื่องที่น่าตกใจคือมันทำงานเกินคำสั่งด้วย ผมสั่งแค่แปลง Schema แต่มันไปตรวจเจอและแก้บั๊คเรื่อง API ที่ผมเขียนไว้ให้เสร็จสรรพ (<-- ถือเป็นเรื่องดี) พร้อมทั้งทำ Dashboard สรุปข้อมูลให้อีกด้วย (<-- จะว่าดีก็ดีแต่มันเปลือง Token ไม่รู้ทำมาทำไม ไม่ได้สั่งให้ทำสักหน่อย)

ตอนนี้ผมเอาขึ้นโปรดักชั่นเรียบร้อยแล้ว ด้วยเหตุนี้โครงการ Kagami Next Generation (ชื่อชั่วคราว) น่าจะเสร็จสมบูรณ์ภายในเร็วๆ นี้ เดี๋ยวอีกไม่นานผมจะเปิดให้ทุกคนใช้งานเลย ตื่นเต้นมาก
