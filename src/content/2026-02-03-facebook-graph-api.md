แชร์ความโง่ของตัวเอง อย่างที่เคยเกริ่นไว้ว่าโปรเจค Kagami Next Generation (ชื่อชั่วคราว) คือการย้ายข้อมูลจากเพจคางามิทั้งสองเพจไปไว้ที่เว็บไซต์ของตัวเอง ซึ่งทั้งหมดมีอยู่ประมาณ 5,000 โพสต์ ใช้แรงงานคนไม่ไหวแน่นอน ต้องหาเครื่องมืออะไรสักอย่างมาช่วยคัดลอกข้อมูล

โปรแกรมดูดข้อมูล (web scraping tool) เสียเงินทุกอันแถมยังไม่ตอบโจทย์ ช่วงปลายปีที่แล้วผมเลยพยายามสร้างบอทและเขียนสคริปต์ขึ้นมาเอง ปัญหาที่ตามมาคือข้อมูลผิดเพี้ยนหรือบางครั้งก็ใช้งานไม่ได้เลย นอกจากนี้ยังไม่นับเรื่องการ clean ข้อมูลที่ยุ่งยากและซับซ้อนพอสมควร

จนกระทั่งผมพึ่งมาค้นพบโดยบังเอิญว่าเฟซบุ๊คมี API ให้ใช้งานอยู่แล้ว พอลองทำตามขั้นตอนแค่แปปเดียวก็ได้ข้อมูลในรูปแบบ JSON ที่เอาไปใช้งานต่อได้ทันที ได้ข้อมูลครบถ้วนสมบูรณ์ทั้งข้อความและรูปภาพ สวยงามมาก

ความรู้สึกตอนนี้เหมือนเดินขึ้นบันไดมาตลอดแล้วพึ่งรู้ว่ามีลิฟต์ให้ใช้ด้วย เอาความพยายามเกือบหนึ่งเดือนครึ่งคืนมาได้ไหมเนี่ย
